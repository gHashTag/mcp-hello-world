# NeuroBlogger MCP Service

Сервис для обработки задач с использованием Model Context Protocol.

## 🚀 Установка

```bash
npm install
```

## 🔧 Конфигурация

Необходимо установить следующие переменные окружения:
- `MCP_SERVER_URL` - URL MCP сервера
- `MCP_API_KEY` - API ключ для доступа к MCP серверу

## 🏃‍♂️ Запуск

```bash
npm run build
npm start
```

## 🧪 Тестирование

### Важно! 
Все новые функции ДОЛЖНЫ быть протестированы в тестовом окружении перед добавлением в основную кодовую базу.

### Подготовка тестового окружения:

1. Убедитесь, что у вас установлен Docker и Docker Compose
2. Запустите тестовое окружение:
```bash
docker-compose -f docker-compose.test.yml up --build
```

### Структура тестов:

- Все тесты находятся в директории `src/test-utils`
- Каждый тест должен:
  - Использовать эмодзи для логирования
  - Следовать формату TestResult
  - Быть самодостаточным
  - Иметь четкие условия успеха/неудачи

### Запуск тестов:

```bash
# Локально
npm run test

# В Docker
docker-compose -f docker-compose.test.yml run test
```

## 📝 Процесс разработки

1. Создайте новую ветку для вашей задачи
2. Напишите тесты в `src/test-utils`
3. Реализуйте функционал
4. Убедитесь, что все тесты проходят в тестовом окружении
5. Создайте PR
